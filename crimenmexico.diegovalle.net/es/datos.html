Data

Todos los datos de este sitio web provienen del SESNSP, además existen datos del sistema de salud para homicidios, pero estos son actualizados con menor frecuencia. Si usan estos datos les agradecería incluyeran un link a la página. En los datos municipales a veces faltan estados porque se tardan en reportar sus datos, si esto pasa pueden esperar un par de días para que esten completos.

Nueva Metodología:
nm-fuero-comun-municipios.csv.gz
nm-fuero-comun-estados.csv.gz
nm-victimas.csv.gz
Vieja Metodología (hasta el 2017):
fuero-comun-municipios.csv.gz
fuero-comun-estados.csv.gz
victimas.csv.gz

Los datos del INEGI están disponibles en un paquete R: mxmortalitydb y en el sitio web del INEGI. Para crear las gráficas usé el paquete R mxmaps.

La información de las bases de datos es constantemente actualizada del sitio web del Secretariado Ejecutivo del Sistema Nacional de Seguridad Pública y del INEGI

La fuente del programa está disponible en GitHub

La fuente de los datos de población es la CONAPO. La población mensual fue obtenida por extrapolación lineal

Los códigos de estados y municipios son del Catálogo de entidades federativas, municipios y localidades del INEGI

Hay datos a nivel estatal y municipal para todos los crímenes en todas sus categorías del fuero común y federal

Los datos de víctimas vienen de los reportes especiales del SNSP del fuero común y de secuestro para el fuero federal
